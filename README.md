## 🛠️ Setup Instructions

### Clone the repository

```
git clone https://github.com/khushboo272/personal_portfolio.git
cd portfolio_website
```

### Install dependencies

```
npm install
```

### Run the project

```
npm run dev
```

Open http://localhost:3000

---

## 🖼️ Assets Guide

Place your assets inside `public/assets/`

| Asset          | Path                                   |
| -------------- | -------------------------------------- |
| Profile Image  | public/assets/profile/profile.jpg      |
| Logo           | public/assets/logo/logo.png            |
| OG Image       | public/assets/og/og-image.png          |
| Project Images | public/assets/projects/<project-name>/ |
